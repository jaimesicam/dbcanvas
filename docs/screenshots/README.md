# Screenshots

Drop PNGs here with these exact names and they'll render in the top-level `README.md`:

| File | Screen | How to capture |
| --- | --- | --- |
| `dashboard.png` | Dashboard | `#dashboard` — with a stack deployed so the live bar charts have data |
| `stack-designer.png` | Database Stacks | `#stack-designer` — a canvas with a few nodes/frames laid out |
| `data-generator.png` | Data Generator | `#data-generator` — a table selected showing the per-column generator template |
| `notifications.png` | Notifications | the top-right bell dropdown open with a few events |

Tips:
- Use the app at `http://localhost:8080` (or your `APP_PORT`) after `make compose`.
- ~1400px-wide window captures look good in the README grid; PNG preferred.
- A dark theme (theme picker, top-right) tends to photograph well.
